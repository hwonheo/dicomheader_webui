DICOM::Header-Web

이 도구는 DICOM 파일들을 자동으로 정렬하여 구조화된 폴더로 정리합니다. 각 DICOM 파일은 SeriesDescription과 SliceThickness에 따라 분류됩니다.

This tool automatically sorts DICOM files into structured folders. Each DICOM file is classified according to its SeriesDescription and SliceThickness.

사용 방법:

How to use:

  1. DICOM 파일이 있는 루트 (최상위) 디렉토리 경로를 입력하세요.
  2. 선택적으로 결과물을 저장할 디렉토리 경로를 지정할 수 있습니다. 비워두면 자동으로 생성됩니다.
  3. '정렬 시작' 버튼을 클릭하여 프로세스를 시작하세요.
  1. Enter the root directory path where DICOM files are located.
  2. Optionally, specify a directory path to save the results. If left empty, it will be automatically generated.
  3. Click the 'Start Sorting' button to begin the process.
